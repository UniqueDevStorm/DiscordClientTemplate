# Discord Client Template (Cogs Structure)

## Test Environment
- Windows 11 Home 22000.613
- Python 3.9.8 64-bit
- discord.py 1.7.3
- jishaku 2.4.0

## Installation
```shell
pip install -r requirements.txt
```
Create an `.env` file.
```dotenv
TOKEN=Your client token
```
## Usage
```shell
python .
```
